# Support

## Getting Help

We're here to help! If you have questions or need support, here are the best ways to reach us:

### 📚 Documentation
- Check the README and documentation in the project repository
- Look at existing GitHub Issues and Discussions
- Review any guides or tutorials in the repository

### 💬 Community Discussions
Visit our [GitHub Discussions](https://github.com/orgs/citizenweather/discussions) to:
- Ask questions about using our tools
- Share ideas and feedback
- Connect with other community members
- Discuss weather data and observations

### 🐛 Reporting Issues
If you've found a bug or something isn't working as expected:
1. Check if the issue has already been reported
2. Provide as much detail as possible:
   - What you were trying to do
   - What happened instead
   - Your environment (OS, version, etc.)
   - Any error messages or logs

### 📧 Direct Contact
For matters that don't fit in public discussions, maintainers can be reached through:
- GitHub Issues with `[private]` in the title
- Discussion posts marked as private

### 🚨 Security Issues
**Do not** report security vulnerabilities publicly. See [SECURITY.md](SECURITY.md) for responsible disclosure guidelines.

---

**We appreciate your interest in Citizen Weather Foundation!** Whether you're contributing code, data, ideas, or just using our tools, you're part of making weather data accessible to everyone. 🌤️
