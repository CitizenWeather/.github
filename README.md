# CitizenWeather Organization

Welcome to CitizenWeather! This is the central configuration repository for our organization on GitHub.

## 📋 What's in This Repository?

This repository contains shared GitHub configuration, templates, and workflows for all CitizenWeather projects:

- **Issue Templates** — Standardized issue types for bug reports, feature requests, and discussions
- **Pull Request Templates** — Guidelines and checklists for contributions
- **GitHub Actions Workflows** — Reusable CI/CD pipelines for testing, security scanning, and releases
- **Documentation** — Contributing guidelines, code of conduct, and security policy
- **GitHub Discussions Templates** — Structured templates for community conversations

## 🚀 Quick Start

### For Contributors

1. Read our [Contributing Guidelines](./CONTRIBUTING.md)
2. Review our [Code of Conduct](./CODE_OF_CONDUCT.md)
3. Check the relevant project's README for setup instructions
4. Use issue templates when creating issues
5. Follow the pull request template guidelines

### For Maintainers

- Use the workflows in `.github/workflows/` for CI/CD automation
- Reference `SECURITY.md` for vulnerability reporting procedures
- Manage organization settings and branch protection rules
