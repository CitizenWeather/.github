# Citizen Weather Foundation Organization

This is the `.github` repository for the Citizen Weather Foundation organization. It contains organization-wide community health files, templates, and policies.

## What's Here

- **[profile/README.md](profile/README.md)** - Organization overview and main projects
- **[CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)** - Community standards and expectations
- **[CONTRIBUTING.md](CONTRIBUTING.md)** - How to contribute to our projects
- **[SECURITY.md](SECURITY.md)** - Security policy and vulnerability reporting
- **[SUPPORT.md](SUPPORT.md)** - Getting help and support
- **[FUNDING.yml](FUNDING.yml)** - Sponsorship information
- **[ISSUE_TEMPLATE/](ISSUE_TEMPLATE/)** - Issue templates for bug reports and features
- **[PULL_REQUEST_TEMPLATE.md](PULL_REQUEST_TEMPLATE.md)** - Pull request template
- **[LICENSE](LICENSE)** - Project licensing (Unlicense/Public Domain)

## Organization Structure

These files apply to all repositories under the Citizen Weather Foundation organization. Individual repositories can override these defaults with their own `.github` directory.
