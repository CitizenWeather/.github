# 🌤️ the Citizen Weather Foundation

> **Open weather data, for everyone.**

We build open-source tools that put weather intelligence in the hands of communities — not just meteorologists. From hyperlocal forecasts to community-driven observation networks, we believe weather data should be accessible, transparent, and free.

---

## What We're Building

| Project | Description |
|---------|-------------|
| 🗺️ **CitizenWeather** | Community weather observation platform with real-time maps |
| 📡 **WeatherNet** | Distributed sensor network firmware for personal weather stations |
| 📊 **OpenSkies API** | Free, open weather data API powered by community observations |
| 🔔 **StormWatch** | Hyperlocal severe weather alerts for your neighborhood |

---

## Why Citizen Weather?

Official weather networks can't be everywhere. A single weather station might cover dozens of square miles — but microclimates are real, and a storm on one block doesn't look the same two streets over.

We're building the infrastructure for a **citizen-powered weather network**: thousands of personal stations, unified into a single open dataset that anyone can query, contribute to, and build on.
